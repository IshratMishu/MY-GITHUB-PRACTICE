MY FIRST DAY AT GITHUB

Github is a web-based hosting platform where we need to create repository. Repository means a folder where we store projects. The repository name and the name of local C drive projects name must be the same, otherwise this will not work.

New repository on the command line are-

  • git init
  • git add .
  • git commit -m "first commit"
  • git branch -M main
  • git remote add again
  • git push -u origin main
